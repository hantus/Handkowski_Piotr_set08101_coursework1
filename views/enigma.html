How does it work?

This cipher is based on enigma machine, developped in Germany in early 20th century by Arthur Sherbius, which was used during World War 2. To use this simplified version:

  • Set the rotors
  • Enter your message
  • Press the Cipher/Decipher button

To Decipher the message:

  • Set the rotors to the same position that was used to encode the message
  • Press the Cipher/Decipher button

You can learn more about enigma machine here.
