ZIP CIPHER

How does it work?

Zip cipher encodes a message by repositioning letters in the message. It cuts provided string in half and zips it together letter by letter form each half. To use it:

  • Enter a message
  • Press the Zip button

To decipher a message

  • Enter the encoded message
  • Pres the Unzip button
